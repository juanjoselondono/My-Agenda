## Usage

Run project with expo

```bash
  git clone repolink
  cd uniagenda
  npm install
  npm run start
```
Generate Apk Build

```bash
eas build -p android --profile preview
```
Generate IOS Build

```bash
eas build -p ios --profile preview
```
Generate .abb Build 

```bash
eas build --platform android```
